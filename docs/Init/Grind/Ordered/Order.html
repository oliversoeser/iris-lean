    class Lean.Grind.Preorder (α : Type u) extends LE α, LT α :
    Type u

    A preorder is a reflexive, transitive relation ≤ with a < b defined in the obvious way.

    • le : α → α → Prop
    • lt : α → α → Prop
    • le_refl (a : α) : a ≤ a
    • le_trans {a b c : α} : a ≤ b → b ≤ c → a ≤ c
    • lt_iff_le_not_le {a b : α} : a < b ↔ a ≤ b ∧ ¬b ≤ a
